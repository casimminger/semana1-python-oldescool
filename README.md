# Semana 1 & 2 - Python + Chatbot Old Es Cool

Proyectos desarrollados durante las primeras dos semanas del roadmap de AI Engineer.

## Proyectos

### 1. Reporte de Inventario (`reporte_inventario.py`)
Script que consume la API de tipo de cambio en tiempo real y genera un reporte CSV con el inventario valorizado en ARS y USD.

**Tecnologías:** Python, requests, csv, API REST

### 2. Chatbot de Ventas (`chatbot_oldescool.py`)
Asistente de ventas conversacional para Old Es Cool con memoria de conversación. Responde consultas sobre productos, marcas, talles, horarios y ubicación del local.

**Tecnologías:** Python, Groq API, LLM (Llama 3.1), manejo de errores, logging

## Negocio
Old Es Cool — local de ropa urbana y streetwear masculino en Nueva Córdoba, Córdoba Argentina.
